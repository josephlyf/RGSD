# RGSD
RGSD is a standard involute tooth profile dataset containing strictly paired static and dynamic blurred images. The sum of images is 4320, the image size is 640Ã—640. All motion parameters, gear parameters, and optical calibration parameters are provided in parameter.txt, which have been verified by precision instruments with only minor deviations.
